# FAQ Chatbot

## Description

FAQ Chatbot is a Python-based application that answers frequently asked questions using NLP-based text matching. The chatbot compares user questions with predefined FAQs and provides the most relevant response through a user-friendly graphical interface.

## Features

* Interactive GUI
* NLP-Based Question Matching
* Instant FAQ Responses
* Clear Chat Functionality
* Enter Key Support
* User-Friendly Interface

## Technologies Used

* Python
* Tkinter
* RapidFuzz

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run the application

python chatbot.py
